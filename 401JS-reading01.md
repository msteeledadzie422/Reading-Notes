# Node Ecosystem

*1. How would you describe Node to a non-technical friend?*

Node is a program that enhances the features of the Chrome V8 engine and allows you to execute JavaScript on your computer.

*2. What does it mean that Node is a JavaScript runtime?*

It allows us to execute JavaScript on our computers.

*3. What is Node used for?*

One of Node's common uses is the installing and running of various build tools designed to automate the process of developing a modern JavaScript application.
